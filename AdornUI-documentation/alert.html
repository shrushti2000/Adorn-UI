Alerts

Alerts are used to force the user to read a specific message which might specify danger,success,warning or certain information to the user. It takes away the attention of the user from current window and diverts him to specific message.

To use this alert, add class alert-box. Include classes alert-success, alert-danger,alert-warning,alert-primary,alert-secondary
Success! Indicates positive action to the user.
Danger! Indicates danger or negative action to the user.
Warning Alert. Indicates warning to the user.
Primary Alert. Indicates any regular or normal information to the user.
Secondary Alert. Indicates any secondary information to the user.
